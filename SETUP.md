# 🚀 Spoonify Setup Guide

## ✅ Quick Start

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Start Development Server:**
   ```bash
   npm start
   ```

3. **Open Browser:**
   - Go to `http://localhost:3000`
